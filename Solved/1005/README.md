# ACM Craft
https://www.acmicpc.net/problem/1005
## 해결 과정
### 0. 평범한 BFS랑 방문 해야하는 횟수로 조절할 수가 없다.
왜냐하면 시간초과가 뜨기 때문이고, 혹은 메모리가 터질 수 있다. 노드 수가 10만개, 간선 수가 10만 - 1개 이다. 
### 1. 위상 정렬로 풀어야 한다.
### 2. 방문해야 하는 횟수 즉, 노드 간의 이어진 것을 '차수'라고 하고, 차수가 0이면 스타트 지점으로 봐도 된다.
차수가 0이 아닌 이 외에는 선행 조건이 있기 때문에 재귀적으로 돌릴 수 없도록 제한두어야 한다.    
### 3. 선행 조건을 만족하면 시간을 기록할 수 있었는데, 이는 DP(메모이제이션)를 암시한다.
### 4. DP에 각 노드들의 건설 시간을 기록하고, 차수가 0인 노드들을 스타트 지점으로 삼고 돌린다.
### 5. Queue 돌리는 과정에서 방문할 노드의 차수 - 1를 해준 뒤 DP[next] = max(DP[next], DP[now] + D[next])를 저장한다.
### 6. 5번에서 차수가 0이 되면 Queue에 next 노드를 푸쉬한다.
### 7. 이를 반복한다.
### 8. Queue 비우면 DP[W]를 호출하면 끝.
