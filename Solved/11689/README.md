### 1. 최대 공약수가 1로 나오게 하려면 a와 b가 서로소 관계임을 나타내야 한다.
### 2. 즉, 크기 * (1 - (1/m)) * (1 - (1/n))로 개수를 알 수 있다고 한다.
### 3. (1 - (1/m))를 따로 구할 땐, 아래와 같다.
i = 2로 스타트를 한 후, 소수를 구하듯이 제곱근 원리를 이용해서 빠르게 구하도록 sqrt(n) 범위 내에서 구하면 된다.		
n를 i로 나누어 떨어지면 n를 n / i로 저장한다.
또 같은 값 i로 나누어 떨어질 수 있다면 for로 나누어 떨어질 수 없을 때까지 n = n / i를 저장한다.
예를 들어 1024 라고 하면 2 ^ 10이고 이 서로소 관계와 오일러 피 함수 공식에 의해서 최대 공약수 1로 만들 수 있는 피변수는 1개밖에 없다. 이 1개는 2를 의미한다.		
또, 81 라고 하면 3 ^ 3 밖에 없으므로, 최대 공약수 1로 만들 수 있는 피변수는 1개 밖에 없다. 이 1개는 3를 의미한다.		
이렇게 되면 (1 - (1 / i1)), (1 - (1 / i2)), ..., (1 - (1 / im))은 각 (1 - (1 / m))가 된다.		
처음에 설정한 answer = n에 각각 곱해주면 된다.		
### 4. (1 - (1/n))를 따로 구할 땐, 아래와 같다.
나누어 떨어질 수 없을 정도로 다 나뉘어졌다면, 마지막으로 n이 1보다 큰지 체크한다. 이 말은 소수로 남아있는지 체크하는 것이다.		
소수는 2, 3, 5, 7, ... 이므로 반복문이 끝날 때 2이상 되어 있다면 소수가 남아있다는 말이 된다.		
그래서 n이 1보다 크면 (1 - (1 / n))로 마지막으로 곱해주면 끝이다.
