# 가장 긴 문자열
https://www.acmicpc.net/problem/3033
## 찾은 해결 방법들
### 0. 각 부분 문자열에 대해서 등장 횟수 2이상이면 그 부분 문자열의 길이를 기록한다.
i = 2 하고, 2부터 N까지 달린다.
### 1. 기록한 것들 중에서 가장 긴 길이를 구한다.
### 2. TLE가 떴다.
### 3. 그리디하게 가장 긴 부분 길이부터 시작해서 2까지 반복 돌리게 한다.
### 4. 등장 횟수가 2이상이면 즉시, 출력하고 종료한다.
### 5. 그래도 TLE가 떴다.
